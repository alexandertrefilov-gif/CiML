# 0003 Visual Style Freeze

Date: 2026-04-26

## Decision

The current visual direction is frozen until a separate design decision is made.

The active visual references are:

- `prototypes/active/ciml_phase1_live.html`
- `prototypes/active/ciml_phase2_after_execution.html`

The reference/legacy prototype is:

- `prototypes/reference/website.html`

## Rationale

The next development work should turn the accepted prototype direction into maintainable WordPress structures. Changing the visual language during architecture work would mix design decisions with implementation decisions.

## Consequences

- Do not redesign typography, color palette, spacing, cards, hero sections, or page composition without explicit approval.
- WordPress implementation should preserve the visual language of the active prototypes.
- Any future design change should be recorded as a new decision.
